-INSTALAR EN MAQUINA NUEVA (Primera ejecución)

·Creas un entorno virtual nuevo:
python -m venv venv
.\venv\Scripts\activate

·Instalas dependencias:
pip install -r requirements.txt



-EJECUTAR

# activar el entorno virtual
.\venv\Scripts\Activate.ps1

# ejecutar tu script
python app.py

**Si queremos ejecutar en 2º plano
  ·lanzamos
    nohup python app.py > output.log 2>&1 &
  ·comprobamos proceso en marcha
    ps aux | grep app.py
    
    ubuntu    12345  0.2  1.5 123456 45678 ?  S  18:12   0:05 python app.py
    ubuntu    12360  0.0  0.0   6432   892 pts/0  S+  18:12   0:00 grep --color=auto app.py

    El PID es la segunda columna → en este caso 12345.
    El otro que ves (grep) lo ignoras.

  ·si queremos matar proceso
    kill <PID>
